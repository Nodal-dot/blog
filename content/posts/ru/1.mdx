Добро пожаловать в мой блог — это мой первый публичный проект, в котором я собрал инструменты и идеи для публикации статей, демонстрации проектов и экспериментов с фронтенд‑технологиями.

Что внутри

- Стек: Next.js (app‑router), React 19, TypeScript.
- Контент: MDX + `gray‑matter` для метаданных — проект‑посты лежат в `content/projects/{locale}` и подгружаются функцией `getProjects`.
- Локализация: `next-intl` и структура папок `src/app/[locale]` для ру/англ версий.
- Стили: Sass + CSS‑модули для изоляции стилей компонентов.
- Интерфейс: готовые UI‑компоненты (`Header`, `Footer`, `ProjectCard`, `ProjectSection`, `ThemeToggle`).

Интересные решения

- `ThemeProvider` управляет тёмной/светлой темой через `data-theme` на `document.documentElement` и `localStorage` — простое и отзывчивое решение для пользовательской темы.
- `PageTransitionProvider` создаёт портальную анимацию загрузки при навигации и аккуратно синхронизирует переход с роутингом — это даёт ощущение «плёнки» между страницами.
- Раздел проектов реализован через компоненты (включая `ProjectSwiper` на базе `swiper`) и читает MDX‑файлы, что позволяет смешивать контент и компоненты в одной записи.

Инструменты и качество кода

- В проекте настроен набор скриптов: `dev`, `build`, `lint`, `storybook`, `test`.
- Инструменты разработки: ESLint, Prettier, Stylelint, Jest/Vitest и Storybook для компонентов.

Архитектура и как запустить
Код организован по компонентам в `src/components`, провайдерам в `src/providers` и утилитам в `src/utils`. Чтобы запустить локально:

```bash
pnpm install
pnpm dev
```
